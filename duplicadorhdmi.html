PARA QUE SERVE O CABO DUPLICADOR SPLITTER HDMI?
-Sua função é apenas “ESPELHAR” a imagem do Monitor/TV para os demais aparelhos de vídeo,
lembrando-se de se atentar a compatibilidade de ambos.
-Conectores banhados a Ouro para uma melhor transferência de sinal.
Permite conectar dispositivos HDMI que Build-in portas HDMI, como HDTV, conversor ascendente DVD Player, monitor de LCD.
-VISANDO QUE: Não funciona de forma inversa, ou seja, não serve para conectar dois aparelhos a uma única TV,
que seria duas entradas e uma saída. Como dito acima ele possui 01 entrada e 02 saídas
ANTES DE REALIZAR A COMPRA SE ANTENTE QUE:
- Este dispositivo não é compatível com aparelhos que tenha o bloqueio de vídeo HDCP

telas auxiliares ele exibira a mesma imagem em todas as telas; não exibe uma tela diferente em cada monitor.
